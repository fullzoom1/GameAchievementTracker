# Менеджер игровых достижений (Achievement Tracker)

Программный модуль для локального учета видеоигр, ведения карточек трофеев и автоматического трекинга времени сессий.

## Технологический стек:
- **Язык**: C# (.NET Framework)
- **Интерфейс**: Windows Forms (WinForms)
- **База данных**: MS SQL Server (ADO.NET)

## Функционал:
- Ведение каталога игр и платформ.
- Автоматический расчет прогресса (ProgressBar).
- Игровой таймер с автоматической триггерной разблокировкой временных ачивок.
